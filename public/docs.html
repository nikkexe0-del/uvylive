uvylive API

A simple REST API for JioTV live channel data — stream URLs, ClearKey DRM credentials, categories, and auth tokens.

REST / JSON
CORS Enabled
Edge Cached
No Auth Required

Base URL

All endpoints are relative to the base URL below.

Production https://uvylive.vercel.app
ℹ️ All responses are application/json with CORS headers set to *. No API key is needed.

Quick Start

Get all JioTV channels in one fetch call.

JavaScript 
const res = await fetch('https://uvylive.vercel.app/api/channels');
const { channels, total } = await res.json();

console.log(`Loaded ${total} channels`);
// → Loaded 312 channels

// Filter by category
const sports = await fetch('https://uvylive.vercel.app/api/channels?category=Sports');
Shell 
# Get all channels
curl https://uvylive.vercel.app/api/channels

# Filter by category
curl "https://uvylive.vercel.app/api/channels?category=Sports"

# Search by name
curl "https://uvylive.vercel.app/api/channels?search=star"
JavaScript (axios) 
import axios from 'axios';

const { data } = await axios.get('https://uvylive.vercel.app/api/channels', {
  params: { category: 'News', limit: 20 }
});

console.log(data.channels);

Usage with Shaka Player 
// 1. Fetch token
const { token } = await (await fetch('/api/token')).json();

// 2. Configure Shaka with ClearKey + token header
player.configure({
  drm: {
    clearKeys: { [channel.keyId]: channel.key },
    advanced: {
      'org.w3.clearkey': {
        headers: { 'cookie': token }
      }
    }
  }
});

Shaka Player

How to play JioTV streams. These are DRM-protected MPEG-DASH streams — Shaka Player is the only browser player that handles ClearKey + auth tokens correctly.

📦 Install: npm install shaka-player — use v4+. Older versions have broken ClearKey handling on Safari/iOS.
HOW Stream flow How it all connects
1. /api/channels→ get url, keyId, key for a channel
2. /api/token→ get session token (auth cookie for Akamai CDN)
3. Shaka Player→ configure ClearKey DRM + inject token into every segment request
4. player.load(url)→ seamless playback, auto-rebuffering on stall
DRM ClearKey configuration Most JioTV channels use ClearKey
🔐 JioTV channels use ClearKey DRM — the key and keyId come from the channel object directly. No license server round-trip needed; everything is in the API response.
ClearKey DRM setup 
import shaka from 'shaka-player/dist/shaka-player.ui.js';

// Install polyfills first — required for iOS/Safari EME
shaka.polyfill.installAll();

const player = new shaka.Player();
await player.attach(videoElement);

// Buffering / retry config — critical for live streams
player.configure({
  streaming: {
    bufferingGoal: 30,          // seconds ahead to buffer
    bufferBehind: 15,           // seconds to keep behind
    retryParameters: {
      maxAttempts: 5,
      baseDelay: 1000,
      backoffFactor: 2,         // exponential backoff
    },
  },
  manifest: {
    retryParameters: { maxAttempts: 5, baseDelay: 1000, backoffFactor: 2 },
  },
});

// ClearKey DRM — keyId and key come from the channel object
player.configure({
  drm: {
    clearKeys: {
      [channel.keyId]: channel.key,  // hex keyId → hex key
    },
    preferredKeySystems: ['org.w3.clearkey'],
  },
});

await player.load(channel.url);
videoElement.play();
AUTH Token injection Akamai CDN auth for segments
🌐 JioTV segment URLs are served via Akamai CDN and require a session token appended as a query parameter. Without it, every segment request returns 403. Inject it via Shaka's network request filter — this runs before every fetch automatically.
Token request filter 
// 1. Fetch the token from our API
const { token } = await (await fetch('/api/token')).json();

// 2. Register a request filter — runs before EVERY network request
player.getNetworkingEngine().registerRequestFilter((type, request) => {
  // Skip DRM license requests — they don't need the token
  if (type === shaka.net.NetworkingEngine.RequestType.LICENSE) return;

  const uri = request.uris[0];

  // Skip data: URIs (inline manifests) and already-signed URLs
  if (!uri || uri.startsWith('data:') || uri.includes('__hdnea__')) return;

  // Append token as query param
  const sep = uri.includes('?') ? '&' : '?';
  request.uris[0] = uri + sep + token;
});
✅ Why this works: The token from /api/token is the raw Akamai auth string (starts with hdnts= or similar). Appending it as a query param on every segment URL satisfies Akamai's CDN authentication check.
JSX React component Drop-in player for any channel
⚛️ Full lifecycle management — creates a player on mount, tears it down on unmount/channel change. The url and clearKeyId/clearKey come straight from the channels API response.
ShakaPlayer.tsx 
import { useEffect, useRef, useState } from 'react';
import shaka from 'shaka-player/dist/shaka-player.ui.js';
import 'shaka-player/dist/controls.css';

interface Props {
  url: string;
  clearKeyId?: string;   // from channel.keyId
  clearKey?: string;      // from channel.key
  token?: string;         // from /api/token
  licenseUrl?: string;   // optional Widevine fallback
}

export default function ShakaPlayer({ url, clearKeyId, clearKey, token, licenseUrl }: Props) {
  const videoRef = useRef<HTMLVideoElement>(null);
  const containerRef = useRef<HTMLDivElement>(null);
  const [error, setError] = useState<string | null>(null);

  useEffect(() => {
    let player: any = null;
    let ui: any = null;
    let mounted = true;

    (async () => {
      shaka.polyfill.installAll();
      if (!shaka.Player.isBrowserSupported()) { setError('Browser not supported'); return; }

      player = new shaka.Player();
      await player.attach(videoRef.current);

      // Attach Shaka UI (built-in controls)
      ui = new shaka.ui.Overlay(player, containerRef.current, videoRef.current);

      player.configure({
        streaming: { bufferingGoal: 30, bufferBehind: 15,
          retryParameters: { maxAttempts: 5, baseDelay: 1000, backoffFactor: 2 } },
        manifest: { retryParameters: { maxAttempts: 5, baseDelay: 1000, backoffFactor: 2 } },
      });

      // DRM: ClearKey (most channels) or Widevine fallback
      if (clearKeyId && clearKey) {
        player.configure({ drm: {
          clearKeys: { [clearKeyId]: clearKey },
          preferredKeySystems: ['org.w3.clearkey'],
        }});
      } else if (licenseUrl) {
        player.configure({ drm: { servers: {
          'org.w3.clearkey': licenseUrl,
          'com.widevine.alpha': licenseUrl,
        }}});
      }

      // Token injection — appended to every segment URL
      if (token) {
        player.getNetworkingEngine().registerRequestFilter((type: any, req: any) => {
          if (type === shaka.net.NetworkingEngine.RequestType.LICENSE) return;
          const uri = req.uris[0];
          if (!uri || uri.startsWith('data:') || uri.includes('__hdnea__')) return;
          req.uris[0] += (uri.includes('?') ? '&' : '?') + token;
        });
      }

      player.addEventListener('error', (e: any) => {
        if (e.detail?.code === 7000) return; // load aborted — ignore
        setError(`Error ${e.detail?.code}`);
      });

      try {
        setError(null);
        await player.load(url);
        if (mounted) videoRef.current?.play().catch(() => {});
      } catch (e: any) {
        if (e.code !== 7000) setError(`Playback error: ${e.message}`);
      }
    })();

    return () => { mounted = false; ui?.destroy(); player?.destroy(); };
  }, [url, clearKeyId, clearKey, token]);

  return (
    <div style={{ position:'relative', aspectRatio:'16/9', background:'#000' }}>
      <div ref={containerRef} style={{ width:'100%', height:'100%' }}>
        <video ref={videoRef} style={{ width:'100%', height:'100%' }} autoPlay />
      </div>
      {error && <div style={{ position:'absolute', inset:0, display:'flex', alignItems:'center', justifyContent:'center', background:'rgba(0,0,0,0.8)', color:'#ff6b6b' }}>{error}</div>}
    </div>
  );
}
Usage 
// Fetch channel + token in parallel
const [{ channels }, { token }] = await Promise.all([
  fetch('/api/channels').then(r => r.json()),
  fetch('/api/token').then(r => r.json()),
]);
const channel = channels[0];

// Render
<ShakaPlayer
  url={channel.url}
  clearKeyId={channel.keyId}
  clearKey={channel.key}
  token={token}
/>
⚠️ iOS/Safari note: shaka.polyfill.installAll() must run before any player is constructed. On iOS, autoplay requires the video element to be user-gesture triggered or muted. Call videoRef.current.muted = true if autoplay fails.

Schemas

TypeScript interface for the Channel object.

TypeScript 
interface Channel {
  id:          string;          // Unique channel identifier
  name:        string;          // Display name
  category:    string;          // e.g. "Sports", "News"
  logo:        string;          // CDN logo image URL
  url:         string;          // MPD/M3U8 stream URL
  keyId?:      string;          // ClearKey DRM key ID (hex)
  key?:        string;          // ClearKey DRM key (hex)
  licenseUrl?: string | null;  // Widevine license URL if applicable
}

interface ChannelsResponse {
  total:    number;
  offset:   number;
  limit:    number;
  channels: Channel[];
}

interface CategoriesResponse {
  total:      number;
  categories: string[];
}

interface TokenResponse {
  token: string | null;
}

StatusMeaningBody

Caching

All endpoints use Vercel Edge Cache via Cache-Control headers.

EndpointCache TTLReason
/api/channels 5 min Channel list rarely changes
/api/categories 10 min Categories are very stable
/api/token 60 sec Auth tokens rotate frequently
